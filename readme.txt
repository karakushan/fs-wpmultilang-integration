Integration layer between the website and WP Multilang functionality for enhanced multilingual capabilities.

== Description ==

FS WP Multilang Integration serves as a bridge between your WordPress website and the WP Multilang plugin, providing enhanced multilingual functionality and seamless integration.

**Features:**
* Enhanced language switching
* Custom content filtering
* Admin integration tools
* Flexible configuration options

**Requirements:**
* WordPress 5.0 or higher
* WP Multilang plugin must be installed and active

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/fs-wpmultilang-integration` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Ensure WP Multilang plugin is installed and active
4. Configure settings through the 'FS Multilang' menu in Settings
