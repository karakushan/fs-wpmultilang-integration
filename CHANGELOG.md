# Changelog

All notable changes to FS WP Multilang Integration will be documented in this file.

## [1.0.0] - 2026-01-29
### Added
- Initial plugin release
- Basic plugin structure and framework
- Language switcher class with core functionality
- Admin settings page placeholder
- Dependency checking for WP Multilang
- Example usage file
- Proper plugin headers and metadata
- Uninstall script for clean removal
- Dedicated hreflang handler class (FS_WPML_Hreflang_Handler)
- URL translator class (FS_WPML_URL_Translator) with full product, post, and category slug translation

### Changed
- Implemented Composer PSR-4 autoloading for class management
- Replaced manual autoloader with Composer vendor autoloader
- Created composer.json with optimized autoload configuration
